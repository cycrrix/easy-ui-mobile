## API

| Name     | Description    | Type                  | Default        |
| -------- | -------------- | --------------------- | -------------- |
| msg      | 弹框文本       | `string`              | `--`           |
| duration | 弹框持续时间   | `number`              | 1500           |
| parent   | 弹框所在父容器 | HTMLElement \|Element | document.title |
